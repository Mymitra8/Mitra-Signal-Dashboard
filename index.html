🚀 Welcome to Mitra Signal

To get started, please enter your Google Apps Script Web App URL. This connects your dashboard to the trading bot backend.

📖 Setup Instructions

1. Go to script.google.com
2. Create a new project and paste the Google Apps Script code
3. Click Deploy → New Deployment
4. Choose "Web App" as deployment type
5. Set "Execute as" to "Me" and "Who has access" to "Anyone"
6. Copy the Web App URL and paste it here
Example URL:
https://script.google.com/macros/s/AKfycbx.../exec
⚠️ Important: Make sure to update your Telegram Bot Token and Chat ID in the Google Apps Script
